Classes summary

IntString

Representa un elemento mixto, compuesto por un entero y una cadena unidos (en ese orden). El método IntString::toString obtiene esa cadena compuesta.
Object Clase base de objetos, para encapsular propiedades y otros métodos básicos.
Type

Represents a PHP object type, and provides some properties and methods to describe some info about itself.
Version

Representa el número de versión de un programa o ensamblado, de la forma "1.2.3.4". Sólo siendo obligatorios el primer y segundo componente. No se puede heredar esta clase.
VersionComponent

Representa un componente de un número de Version. Extiende la clase IntString, pero restringe los valores que puede tomar.
